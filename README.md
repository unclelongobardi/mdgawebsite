# mdgawebsite

Sitio web estático servido con un pequeño servidor Node.js.

## Estructura

- `index.html` — Página principal
- `server.js` — Servidor HTTP estático simple (Node.js, sin dependencias)
- `backgroundimage/` — Imágenes de fondo y otros recursos estáticos

## Cómo ejecutar localmente

Requiere Node.js instalado.

```bash
node server.js
```

Luego abre [http://localhost:8000](http://localhost:8000) en tu navegador.
